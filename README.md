# афе 'орошек' - еб-сайт

то проект веб-сайта для семейного кафе 'орошек', разработанный с использованием [Next.js](https://nextjs.org).

## ункциональность

- **лавная страница:** риветствие, преимущества, призыв к действию.
- **еню:** росмотр блюд (в разработке).
- **орзина:** правление заказом (в разработке).
- **ронирование:** орма бронирования столика (в разработке).

## ачало работы

ля запуска сервера разработки выполните:

`ash

``n
ткройте [http://localhost:3000](http://localhost:3000) в браузере, чтобы увидеть результат.

## Структура проекта

- \src/app\: Страницы и макеты (App Router).
- \src/components\: ереиспользуемые компоненты (Navbar, Footer и др.).
