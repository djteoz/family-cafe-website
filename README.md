# Семейное Кафе "Уют"

Веб-сайт для семейного кафе, разработанный на [Next.js](https://nextjs.org).

## Функционал

- **Главная страница:** Приветствие, преимущества, отзывы.
- **Меню:** Каталог блюд с фильтрацией по категориям и тегам.
- **Корзина:** Добавление товаров, изменение количества, подсчет суммы.
- **Бронирование:** Форма бронирования столика с отправкой уведомлений в Telegram.

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Создайте файл `.env.local` и добавьте настройки Telegram (см. `.env.local.example` или инструкцию):

```
TELEGRAM_BOT_TOKEN=ваш_токен
TELEGRAM_CHAT_ID=ваш_id
```

3. Запустите сервер разработки:

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Структура

- `src/app`: Страницы и роутинг (App Router).
- `src/components`: Компоненты (Navbar, Footer и др.).
- `src/context`: Управление состоянием (Корзина).
- `src/data`: Данные меню.
